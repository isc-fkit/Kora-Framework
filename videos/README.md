# videos/

Đặt file hướng dẫn vào đây nếu muốn host cùng repo, vd `guide.mp4`,
rồi sửa trong `index.html`: `const VIDEO = { youtube:"", file:"videos/guide.mp4" };`

Lưu ý: GitHub giới hạn 100MB/file và băng thông Pages có hạn — video >50MB nên
đưa lên YouTube rồi dùng `youtube:"MÃ_VIDEO"` thay vì file.
