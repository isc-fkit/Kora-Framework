Người dùng vừa CHỦ ĐỘNG gõ `/kora-nap-file` — đây là **lệnh rõ ràng** để nạp tài liệu thành tri thức.

- Nếu user CHƯA đính kèm file → nhắc user **kéo file vào chat** (PDF / DOCX / PNG / JPG / zip
  Obsidian), chờ có file rồi mới chạy.
- Khi đã có file → đọc và thực thi `workflows/02-import-files.md` theo `CLAUDE.md`.
- Giữ nguyên Approval Gate: chỉ ghi vào `docs/` sau khi user approve.
- Sau khi nạp xong, đề xuất bước kế gồm cả lựa chọn **nạp thêm nguồn khác** (quét Jira / nạp
  tài liệu khác) theo §0.4.
