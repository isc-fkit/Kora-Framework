# Domain Rules — rule nghiệp vụ theo domain (PHẦN ĐỘNG)

> File này được tạo từ preset lúc setup và user có thể sửa bất cứ lúc nào
> (nhắn "đổi domain" hoặc "sửa rule"). Claude đọc file này trước MỌI phiên phân tích.

**Trạng thái: CHƯA SETUP.** Chạy `@khởi tạo dự án` để chọn domain — Claude sẽ copy
preset từ `config/domain-presets/` vào đây rồi tùy biến theo mong muốn của bạn.

## Cấu trúc rule (sau khi setup file sẽ theo dạng này)

### 1. Thuật ngữ & quy ước
### 2. Rule phân tích nghiệp vụ
### 3. Rule thận trọng (những gì AI không được tự suy diễn)
### 4. Design Rules (ràng buộc UI/UX theo domain)
### 5. Code Rules (ràng buộc kỹ thuật theo domain)
### 6. Rule tài liệu & ngôn ngữ
