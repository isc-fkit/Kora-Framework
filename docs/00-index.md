# Mục lục Knowledge Base (cho người đọc)

| Thư mục | Nội dung |
|---|---|
| 01-domain/ | Tri thức nghiệp vụ domain: bối cảnh, quy định |
| 02-product/ | Sản phẩm: personas, tổng quan tính năng, hành trình người dùng |
| 03-features/ | Mỗi tính năng một folder F-xxx (source/ cho Claude, export/ cho người đọc) |
| 04-design/ | Design system, prototype map, design notes |
| 05-architecture/ | Kiến trúc hệ thống, mô hình dữ liệu, tích hợp |
| 06-decisions/ | ADR — nhật ký quyết định |
| 07-research/ | Tài liệu nghiên cứu, tham khảo (gồm master workflow spec) |
| 08-glossary/ | Thuật ngữ, từ viết tắt |

Mục lục máy đọc: `.kb/index.json`. Lịch sử thay đổi: `.kb/changelog.md`.
