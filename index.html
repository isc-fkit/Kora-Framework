Cài đặt và dùng 12 lệnh /kora-* để nạp tri thức từ Jira & tài liệu, tự phân tích và sinh báo cáo — ngay trong Claude.

Dán 1 dòng vào Terminal

Cài sạch vào ~/.claude — không để lại thư mục source. Cách này không bao giờ bị macOS Gatekeeper cảnh báo.

Terminal — macOS / Linux
bash <(curl -fsSL https://raw.githubusercontent.com/isc-fkit/Kora-Framework/release/install.command)

Mở Claude rồi gõ /kora-init

Skill đã nằm trong ~/.claude/commands. Mở một phiên chat mới, gõ / để thấy đủ 12 lệnh /kora-*.

Cập nhật & gỡ

Chạy lại đúng lệnh trên = cập nhật (tự kéo bản mới, tự thêm skill mới). Gỡ sạch:

Terminal — gỡ cài đặt
bash <(curl -fsSL https://raw.githubusercontent.com/isc-fkit/Kora-Framework/release/uninstall.command)
🔑
Nếu kết nối nguồn bằng API token, token được lưu thành biến môi trường trong ~/.zshrc / ~/.bashrc — không lộ trong source hay chat.

Tải & giải nén → double-click

Không quen Terminal? Tải file .zip → giải nén (Finder tự bung) → nhấn đúp install.command — script tự cài skill vào ~/.claude.

macOS / Linux

Tải install.command.zip → giải nén

⬇ install.command.zip

Windows

Tải install.bat

⬇ install.bat

Nếu trình duyệt mở ra dạng chữ thay vì tải → chuột phải nút → “Lưu liên kết thành…” (Save Link As). Hoặc dùng Cách 1 (dán 1 dòng vào Terminal) — không cần tải file.

⚠️
macOS — lần đầu mở (cấp quyền chạy app): double-click install.command (đã giải nén). Nếu bị chặn → vào System Settings → Privacy & Security → kéo xuống mục Security → bấm “Open Anyway” → xác nhận bằng Touch ID / mật khẩu → double-click lại. (Hoặc chuột phải file → Open → Open.)  Windows: “More info” → “Run anyway”. Sau lần đầu script tự gỡ nhãn → lần sau chạy thẳng.
🔧
Báo “could not be executed because you do not have appropriate access privileges”? Đó là do tải file .command trực tiếp (mất quyền chạy). Hãy tải bản .zip ở trên (giải nén giữ nguyên quyền chạy), hoặc dùng Cách 1 (curl|bash) — không cần file.

Mở Claude App / Cowork → gõ /kora-init

Cập nhật & gỡ

Chạy lại file cài đặt = cập nhật. Gỡ: gõ /kora-uninstall trong Claude (gọn nhất), hoặc tải uninstall.command.zip / uninstall.bat rồi chạy.

Bắt đầu — tạo project

Sau khi cài skill: trong Claude Cowork mở (hoặc tạo) một folder trống cho project → gõ /kora-init. Nó tự dựng project (docs/ + vault + config) rồi hỏi domain, rule nghiệp vụ, tên project và nơi lưu vault. Init gọn nhẹ — không hỏi token/lịch (việc đó ở /kora-scan, /kora-daily-report, /kora-schedule).

Kết nối nguồn & điền API key

Khi cần kéo dữ liệu, Kora hỏi chọn phương thức:
  • MCP — bật Connector trong Claude App/Cowork (Atlassian/Jira, SharePoint). Không cần token tay.
  • API — dùng Personal Access Token; Kora tự lưu vào ~/.zshrc.
Lấy token (chỉ khi dùng API):
  • Jira Server/DC: Profile → Personal Access Tokens
  • Atlassian Cloud: id.atlassian.com → API tokens
  • SharePoint: qua MCP connector (không cần token tay)
Tự điền vào ~/.zshrc rồi source ~/.zshrc (hoặc mở terminal mới):
~/.zshrc
export KORA_JIRA_BASE_URL="https://jira.cong-ty.vn"
export KORA_JIRA_TOKEN="dán-token-cua-ban"
