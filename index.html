2

Khởi tạo dự án (1 lần)

Trong ô chat Cowork, gõ lệnh khởi tạo. Hệ thống hỏi từng bước (domain, đường dẫn vault…) — bạn chỉ việc chọn.

/kora-khoi-tao        # hoặc gõ: @khởi tạo dự án

Mỗi bước đều dừng hỏi xác nhận — không tự quyết thay bạn.

5

Cập nhật & Sao lưu

Cập nhật chương trình lên bản mới nhất mà giữ nguyên tri thức. Có 2 cách: gõ lệnh trong Cowork, hoặc double-click file script theo hệ điều hành.

macOS / Linux

Tải .command → double-click → Terminal tự mở

Windows

Tải .bat → double-click → CMD tự mở
💡 Lần đầu tiên mở file, hệ điều hành sẽ hỏi xác nhận 1 lần (macOS: chuột phải → Open hoặc Privacy & Security → Open Anyway; Windows: More info → Run anyway). Sau lần đó, script tự gỡ nhãn cho các file còn lại — những lần sau chạy thẳng, không hỏi lại.

Hoặc gõ thẳng trong Cowork:

/kora-cap-nhat        # tự so phiên bản, giữ nguyên tri thức
# macOS/Linux — không tải file nên không bị cảnh báo bảo mật
bash <(curl -fsSL "https://raw.githubusercontent.com/isc-fkit/Kora-Framework/release/scripts/update.command")
✅ Cập nhật xong — tri thức của bạn được giữ nguyên 100%.

Sao lưu / chuyển máy: /kora-sao-luu đóng gói tri thức ra kora-kb-*.zip; máy mới gõ nhập tri thức để khôi phục.
